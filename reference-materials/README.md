# Reference materials

Place official free AAMC PREview practice PDFs here only for private developer
review. Do not commit official PDFs or copy their scenarios, response choices,
rationales, or answer keys into the app.

Recommended private review process:

1. Download the official free AAMC PREview practice exams and any scoring keys
   or rationales from AAMC.
2. Review the scenario length, number of response options, rating distribution,
   rationale tone, and competency mix.
3. Update only high-level prompt guidance, scoring calibration notes, or quality
   checks. Do not include official AAMC PDF content in the app. Use it only as
   private reference material during development to understand format, reasoning
   style, and difficulty.
4. Keep generated and preloaded app scenarios fully original.
